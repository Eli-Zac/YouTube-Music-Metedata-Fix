# About This Script
This Tampermonkey userscript ensures that YouTube Music always provides full, accurate track metadata to your system and extensions. It fixes issues where Windows media controls and tools like [Web Scrobbler](https://webscrobbler.com/) only see partial track information.
### Features
- Shows the full track title, including featured artists (e.g., `Song (feat. Artist)`).
- Correctly sets the main artist.
- Sets the album metadata, even if Windows doesn’t display it.
- Keeps [Web Scrobbler](https://webscrobbler.com/) and other media extensions fully in sync.
- Works automatically with each track change.
- Safe and lightweight, with no polling spam.
